:::{note}
**Keyboard shortcuts** in the HTML slides:
- `→` / `Space` — next slide
- `←` — previous slide
- `F` — fullscreen
- `Esc` — slide overview
- `?` — all shortcuts

**To save as PDF:** click *Download PDF* — the slides open in print layout. Use your browser's **File → Print → Save as PDF** (or `Cmd+P` / `Ctrl+P`).
:::
